Concurrent

mengerjakan secara bersamaan dalam satu waktu

misal, 2 task dilakukan beriringan, berjalan di 1 core CPU, dan dapat berjalan secara beriringan, tergantung pada arsitektur sistem

Parallel

Membagi task menjadi sub-task yang lebih kecil, dapat diproses secara parallel, misal: menjalankan pada multiple CPU dalam wkatu bersamaan
